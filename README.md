<h2> Welcome to my first project! This git repository contains several math games.</h2>

<h1>Minimal requirements</h1>
<ol>
    <li>Linux</li>
    <li>NodeJs</li>
    <li>npm</li>
</ol>

<h1>Installation</h1>
<ol>
    <li>git clone</li>
    <li>make install</li>
</ol>
